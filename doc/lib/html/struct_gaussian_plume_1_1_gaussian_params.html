Detailed Description

The intermediary structure for matrix calculations.

Parameters

windSpeed- wind speed in kmph

x- start of matrix fragment
y- end of matrix fragment
z- air pollution concentration height
xs- a double that is part of the equation calculated based on azimuth (x axis)
ys- a double that is part of the equation calculated based on azimuth (y axis)
H- the height of the emitter
weatherStab- the value representing the atmospheric stability class
isLast- the value for checking the last emitter
