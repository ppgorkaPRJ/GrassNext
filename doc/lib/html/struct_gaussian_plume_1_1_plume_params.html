Detailed Description

The structure PlumeParams that contains data necessary for calculating air pollution.

Parameters
roadStartLat- latitude of the starting point of the road segment
roadStartLon- longitude of the starting point of the road segment
roadEndLat- latitude of the ending point of the road segment
roadEndLon- longitude of the ending point of the road segment
windSpeed- wind speed in kmph
windDirection- the direction of wind in degrees
pollution- type of pollution chosen by the user
emittersHeight- the height on which emitters were installed
weatherStability- the value representing the atmospheric stability class
resolution- the size of one side of the calculated area (in meters)
matrixSize- the size of the matrix
matrixSizeStart- the starting index of the matrix representing calculater area
matrixSizeEnd- the ending index of the matrix representing calculater area
concentrationHeight- the height at which the concentration of air pollution will be calculated
thresholds- a value of a singular treshold
thresholdList- list of tresholds for the matrix
divMatrixSide- the number, by which the matrix will be divided into smaller chunks for parallelization
mean- a variable for storing mean of the data
stdDev- a variable for storing standard deviation of the data
aninstance of the Vehicle class with data read from backend server
gpv- a vector structure containing parameters necessary for Gaussian Plume calculation for each emitter
